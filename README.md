# HydraHive2 Module Hub

First-party module hub for [HydraHive2](https://github.com/hydrahive/hydrahive2.0).
Each top-level directory is an installable module; `hub.json` is the index.

## Modules
- **example** — Minimal walking-skeleton template (notes UI ↔ API ↔ DB). Copy it to author new modules.
